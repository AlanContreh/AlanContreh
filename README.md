## 📱 Projects

### [Obsesive – Workout Tracker](https://play.google.com/store/apps/details?id=com.obsesivelabs.obsesive)

A free, science-based fitness app that analyzes your training routine with a 5-metric scoring system (0–100).

- Routine scoring & post-session summaries
- Offline functionality
- Exercise video tutorials (3D animated)
- Routine sharing via codes
- Multilingual: EN / ES / PT-BR
